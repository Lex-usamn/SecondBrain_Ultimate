💡 PADRÕES DE CODIFICAÇÃO (Regras para Nova IA)
⚠️ ERROS COMUNS A EVITAR:
python

# ❌ NUNCA FAZER ISSO (causa AttributeError):
valor_none.lower()  # Se valor_none for None, CRASH!

# ✅ SEMPRE FAZER ISSO (seguro):
(valor_none or '').lower()  # Funciona mesmo se None

# ❌ NUNCA ISSO (causa ValueError em dataclasses):
alerts_by_level: Dict = defaultdict(int)

# ✅ SEMPRE ISSO:
alerts_by_level: Dict = field(default_factory=lambda: defaultdict(int))

# ❌ CUIDADO COM ENUMS:
Status.UNKNOWN  # Pode não existir!

# ✅ VERIFICAR PRIMEIRO:
Status.READY    # Usar valores que existem no enum
